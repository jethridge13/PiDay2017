This website was made after viewing Matt Parker's Pi Day 2017 video on using random numbers to estimate Pi.

To see the proof and a much more in-depth explanation, click the link above.

The quick version: The probability of two random numbers being coprime is 6/pi^2. For a set of random number rolls, the ratio of coprimes to cofactors can be denoted as X. This means that (approximately) X = (6/pi^2). Using basic algebra, we can rearrange this to get pi = sqrt(6/x).

In conclusion, we can use a bunch of random numbers to estimate Pi.
